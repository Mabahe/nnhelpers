=======================================
die
=======================================

Description
---------------------------------------

<nnt3:die />
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

Macht nichts, außer das Script zu beenden.

Damit kann während des Rendering-Prozesses von Fluid das Script abgebrochen werden.
Praktisch, um z.B. Mail-Templates zu debuggen.

.. code-block:: php

	{nnt3:die()}

| ``@return death``
